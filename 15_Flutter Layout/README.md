## Flutter Layout
 Flutter Layout adalah untuk mengatur tata letak elemen-elemen UI pada sebuah aplikasi Flutter. Flutter menyediakan sejumlah widget yang dapat digunakan untuk membuat layout yang berbeda-beda sesuai dengan kebutuhan aplikasi kita. Layout berbentuk widget yang mengatur widget di dalamnya.

### Single-child Layout
 - Container: Widget ini digunakan untuk mengatur tata letak sebuah widget di dalam kotak atau wadah. Container memiliki properti seperti padding, margin, color, dan lain-lain untuk menyesuaikan tampilan dari sebuah widget.
 - Center: widget yang digunakan untuk menempatkan sebuah widget di tengah-tengah dari area yang tersedia. Widget Center ini dapat digunakan untuk menempatkan widget apapun, seperti gambar, teks, atau widget lainnya, dan akan menempatkannya tepat di tengah-tengah area yang tersedia.
 - Sizedbox: widget digunakan untuk membuat sebuah box dan membungkus widget lain.

### Multi-child Layout
 - Row dan Column: Widget ini digunakan untuk mengatur tata letak widget dalam bentuk baris dan kolom. Row digunakan untuk mengatur tata letak widget secara horizontal, sedangkan Column digunakan untuk mengatur tata letak widget secara vertikal.
 - ListView: Widget ini digunakan untuk mengatur tata letak widget dalam bentuk daftar atau list. ListView memiliki beberapa jenis seperti ListView.builder, ListView.separated, dan lain-lain.
